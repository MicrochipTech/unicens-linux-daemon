UNICENS V2 Daemon (unicensd)
============================

!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
!!! Only use Firmware V2.4.0 or newer                                                  !!!
!!! Make sure that you enable UNICENS mode in ConfigString                             !!!
!!! Make sure that you set an unique node address in IdentString                       !!!
!!! Make sure that you set an unique MAC address in IdentString, when device use MEP   !!!
!!! Use default config feature from MOST Linux Driver to setup USB/MLB.                !!!
!!! Control Channel should be mapped to CDEVs: /dev/inic-usb-crx and /dev/inic-usb-ctx !!!
!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
